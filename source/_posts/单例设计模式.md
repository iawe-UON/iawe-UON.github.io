单例设计模式含义是在程序中使用单例模式设计的类只能有唯一一个实例，并且提供一个全局接口供其他程序访问。

常用场景：
数据库连接池、日志系统

实现方式1：使用import导入实例

python中这是最容易实现的方式

```python
class db_connect:
    def __init__(self):
        self.connect_string = "XXXXXX"
        print(f"数据库连接池创建完成，地址：{self.connect_string}")
    def query(self):
        print(f"正在查询地址为{self.connect_string}的数据库")

db_connection = db_connect()

"""
其他脚本：
import db_connection
db_connection.query()
"""
```
通过import导入可以确保db_connection全局只能被实例化一次
